Nmap OS Detection Script

This Lua script uses the nmap command with the -O option to detect the operating system of a target host. It does this by executing the nmap command on the specified host and parsing the output for the detected operating system information.

Usage

To use this script, first ensure that you have Nmap and the Nmap Scripting Engine installed on your system. Then, save the script as a file with the extension .nse. You can then run the script using the following command:

nmap  --script <path/to/script.nse> target(s)>

Replace <target(s)> with the ports and targets you want to scan, respectively. You can also use the -A option to enable aggressive scanning, which can improve OS detection accuracy. Here's an example:

nmap --script osd.nse example.com
